학습 자료 분석 챗봇

📚 사용 안내

1. 이미지 업로드

• 교과서나 강의 노트의 이미지를 업로드하세요.
• 텍스트가 선명하게 보이는 이미지를 사용하세요.

2. 자동 분석

• 이미지 업로드 후 자동으로 분석이 시작됩니다.
• 핵심 개념, 문제, 암기 카드가 생성됩니다.

3. 추가 질문

• 생성된 내용에 대해 추가 질문을 할 수 있습니다.
• 예: "이 개념에 대해 더 자세히 설명해줘"

4. 응답 형식

• 📚 핵심 개념: 중요 내용 요약
• ❓ 학습 문제: 이해도 확인 문제
• 📝 암기 카드: 암기용 요약

⚠️ 주의사항

• 이미지의 텍스트가 선명해야 정확한 분석이 가능합니다.
• 한 번에 하나의 이미지만 분석 가능합니다.
